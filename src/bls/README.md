Reminder on how to set up a python virtual environment:

- In the powershell terminal:
  - py -m venv .venv
- VS Code:
  - Python: Select Interpreter > Select a version of python
  - Run py .venv/Scripts/activate
  - Add python environment to the workspace folder when prompted
  - Select the .venv environement when prompted after running the first code chunk
  - Install ipykernel when prompted
